# DOMDOM AI v2.6 Kurulum Rehberi

1. Gerekli kütüphaneleri kur:
   pip install -r requirements.txt

2. start_ai.py dosyasını çalıştırarak sistemi başlat:
   python start_ai.py

3. config.json içinden API anahtarlarını gir.

4. Telegram ve WhatsApp entegrasyonu için ilgili modülleri düzenle.

Sistem otomatik öğrenme, al-sat, haber analizleri ve sohbet işlevlerini içerir.
